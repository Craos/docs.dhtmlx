lock

locks the specified cell(s)

void lock(string cell);

Parameters

cellstringthe id of a cell(s) or a range of cells

Example

// locks a cell
spreadsheet.lock("A1");
 
// locks a range of cells
spreadsheet.lock("A1:C1");
 
// locks specified cells
spreadsheet.lock("A1,B5,B7,D4:D6");
