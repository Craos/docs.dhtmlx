ItemPrerender

The ItemPrerender event occurs after data has been selected from the database but before its outputting to client.

connector.ItemPrerender += new EventHandler<ItemPrerenderEventArgs<dhtmlxTreeDataItem>>(connector_ItemPrerender);
 
void connector_ItemPrerender(object sender, ItemPrerenderEventArgs<dhtmlxTreeDataItem> e)
{
   if (e.DataItem.DataFields["is_hidden"] == "1")
   e.DataItem.LeafImage = "lock.gif";
}

In this event the connector_ItemPrerender method receives reference to the data item object currently being rendered. This object has different methods for different client-side components.
