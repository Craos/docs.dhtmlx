DHTMLX 5.1 - Full List of Changes

Major changes

  • TypeScript support for all components
  • promise API for Ajax-requests (all components)
  • JSON data loading for all components
  • JSON-REST data saving with DataProcessor

Updates

  • common - load methods of data components return promise
  • form - the loadStructHTML API for the Form
  • formuploader - the onBeforeFileUpload event
  • grid - a new row operation getRowData API
  • grid - getCellExcellType API
  • grid - better text size adjusting
  • grid - the ability to pass a config object to the attachGrid() method
  • treeview - onClick, onDblClick events for TreeView
  • treeview - showItem API
  • window - a window triggers onResizeFinish after min|max operations

Fixes

  • calendar - a week number for the last week of the year
  • calendar - onChange event for a Calendar in the Toolbar
  • cell - invalid processing of detached URLs
  • chart - memory leaks in the Chart
  • combo - autocomplete for Combo in the Grid
  • common - onXLE and onXLS event for loadStruct()
  • datastore - regression in ID sync between a DataStore and a Scheduler
  • dataview|list - multiselect for MacBook
  • form - getInput() for non-input types
  • form - getItem() works for a checkbox now
  • grid - combo cell in a split Grid
  • grid - Combo in Grid while loading data by POST
  • grid - correct getValue() for an empty "dyn" cell
  • grid - deleting a row with checked radio button in a Grid
  • grid - dynamic data loading in the JSON mode
  • grid - filters and null values in JSON data
  • grid - Grid in the split multiline mode and filters
  • grid - locked rows in the split mode
  • grid - memory leak in dhtmlxGrid
  • grid - options for a Grid synced to a DataStore
  • grid - selectBlock() function fix
  • grid - sorting of the co/coro columns
  • grid - tabbing in the Grid with block selection
  • grid - x-scrolling in Google Chrome
  • layout - tooltip for Layout's cells
  • menu - hide Context Menu after click in the Grid
  • menu - using Context Menu with DOM objects
  • menu - using icons in Menu with inline data
  • popup - Grid in the Layout attached to a Popup. Row selecting problem.
  • toolbar - clearAll() and spacer in Toolbar
  • treegrid - select filters and options from Tree cell
  • treeview - clearAll() in TreeView
  • treeview - drag-n-drop in the zoomed state
  • treeview - folder icon after deleting all child items
  • treeview - missed method deleteChildItems() in the TreeView API
  • window - regression in Context Menu behavior on IE
