StartGantt APIProperties

tooltip_timeout

sets the timeout in milliseconds before the tooltip is displayed for a task

number tooltip_timeout;

Example

gantt.config.tooltip_timeout = 50;
gantt.init("gantt_here");


Default value:

30
Details

This option is defined in the tooltip.js extension, so you need to activate the tooltip plugin. Read the details in the Tooltips for Gantt Elements article.
