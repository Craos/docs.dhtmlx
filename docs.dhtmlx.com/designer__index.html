Visual Designer

Visual Designer is an online tool that allows building user interfaces for web applications using DHTMLX components. You can specify the interface layout, place desired components into it and set their configuration. What is more, it is possible to add corresponding event handlers that will process various actions which happen during interaction with the user interface.

Visual Designer includes an embedded Form Designer that makes it easy to build a form and customize its appearance to achieve the best result.

You can download the source code of the ready application prototype to make some improvements when needed.

The current manual refers to the version 1.0.
