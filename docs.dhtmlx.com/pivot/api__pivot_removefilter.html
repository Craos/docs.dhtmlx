removeFilter

removes a filter and refreshes Pivot

void removeFilter(string field);

Deprecated

use instead:

  • setFilterValue(field_id,{}) to remove a particular filter
  • clearFiltersValues to remove all used filters

Parameters

fieldstringthe ID of a field

Example

pivot.removeFilter("form");
