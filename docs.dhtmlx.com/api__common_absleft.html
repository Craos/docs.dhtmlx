absLeft

returns left element position related to window

number absLeft(string|number|HTMLElement obj);

Parameters

objstring|number|HTMLElementobject or id

Returns

numberleft element position

Example

var t = document.getElementById("someId");
var x = window.dhx.absLeft(t);
 
// or
var x = window.dhx.absLeft("someId");
