Carousel

dhtmlxCarousel is an easy-to-use and handy component intended for displaying various content, including other DHTMLX components on one page. It allows switching cells with the help of special arrow buttons. There are several predefined skins and nice animation effects available for users. The component can be easily customized according to provided requirements by means of rich API.
