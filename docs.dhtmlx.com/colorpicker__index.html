ColorPicker

dhtmlxColorPicker component is a smart color picker that can be easily used either as an inline control or as a popup dialog box.

Its intuitive interface, flexibility and simplicity of usage make this component a good aid for different projects and applications.

ColorPicker gives users the possibility to specify colors in RGB, HSL and Hex formats. Complete JavaScript API is provided to help users to save time on configuring.
