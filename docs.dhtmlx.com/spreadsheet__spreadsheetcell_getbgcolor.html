getBgColor

Gets the background color set in the cell.

Return value

  • (string) the color in the hexadecimal format (without '#')

Example

$cell = new SpreadSheetCell($res, "1","B1", $db_prefix);
$color = $cell->getBgColor();

See also: setBgColor, setStyle
