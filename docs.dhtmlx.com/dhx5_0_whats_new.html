DHTMLX 5.0 - Full List of Changes

Major changes

  • Font Awesome support added for accordion/menu/ribbon/toolbar/treegrid
  • a new component dhtmlxTreeView added (docs, demos)
  • a new Material skin added
  • window.dhx variable (instead of window.dhx4)

Updates and fixes

  • calendar - ability to show calendar next to input
  • chart - origin 0 for stackedBar added
  • combo - html rendering removed for image and checkbox
  • combo - regex parsing fixed for filtering
  • combo - unsync and unbind for dhtmlxCombo fixed
  • editor - getContent fixed for Chrome
  • form - validation fixed for type:radio
  • grid - column styling in header area with colspans fixed
  • grid - combo in grid and non-xml data types fixed
  • grid - correct row height for split grid initialized from HTML
  • grid - correctly escaped quote in the CSV parser
  • grid - custom modes for dataprocessor improved
  • grid - memory leak during column resize fixed
  • grid - mouse wheel scrolling in split grid fixed
  • grid - the onCheck event for master checkbox fixed
  • grid - position of elements in open subrow fixed
  • grid - radio buttons in split grid fixed
  • grid - selectBlock API added
  • grid - sizing in split mode fixed
  • grid - styling of grid header during drag-n-drop fixed
  • grid - subrows can't corrupt userdata serialization
  • grid - using adjustColumnSize in grid with colspans and rowspans fixed
  • grid - using custom skins in split mode fixed
  • layout - attachSidebar fixed for terrace skin
  • list - missed scroll in attached list fixed
  • message - glitch in skyblue skin fixed
  • message - styling improved
  • popup - keep popup visible on row select if grid attached to a form inside
  • sidebar - autohide animation fixed
  • tabbar - onload attribute for api-init fixed
  • tabbar - tab close button fixed
  • toolbar - clicks on touch-devices fixed
  • tree - data serialization in IE11 fixed
  • tree - lockItem doesn't block item opening fixed
