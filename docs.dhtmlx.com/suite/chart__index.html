Chart

dhtmlxChart is a great tool for creating powerful charts for web applications. It provides a wide variety of chart types, the possibility to render several data properties on the same chart and adding tooltips, both 2D and 3D presentation, different variants of data loading, and a whole kit of configuration settings for all elements of a chart interface. Check online samples for dhtmlxChart.
