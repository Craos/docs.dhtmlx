beforeEditEnd

fires before editing of a cell is ended

boolean beforeEditEnd(string value,object row,object column){ ... };

Parameters

valuestringthe new value of a cell
rowobjectan object with a row configuration
columnobjectan object with a column configuration

Returns

booleanfalse - to prevent closing of an editor, otherwise true

Example

grid.events.on("BeforeEditEnd", function(value,row,column){
    // your logic here
    return false;
});
