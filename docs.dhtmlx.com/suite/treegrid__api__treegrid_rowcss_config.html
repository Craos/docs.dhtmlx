rowCss

sets style for a row

function rowCss;

Example

<style>
    .my_custom_row {
        background: coral;
    }
</style>
 
var grid = new dhx.Grid("grid_container", {
    columns: [// columns config],
    rowCss: function (row) { return row.custom ? "my_custom_row" : "" },     data: dataset
});

Related samples

  • TreeGrid. Custom Row Style
Details

The function takes the id of a row as a parameter and returns a string with the name of a CSS class.
