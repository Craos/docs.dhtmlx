Migration from 2.5 to 3.0

This article will help you to migrate from the previous version 2.5 of dhtmlxVault to the totally renewed version 3.0. Check the list below to explore all the changes.

2.5 -> 3.0

The API of version 2.5 is still available, but it is incompatible with the API of version 3.0.

Changed API

  • vault.addFileRecord -> vault.data.add
  • vault.attachEvent -> vault.events.on
  • vault.clear -> vault.data.removeAll
  • vault.detachEvent -> vault.events.detach
  • vault.load -> vault.data.load
  • vault.setAutoStart -> vault.uploader.config.autosend
  • vault.setStrings -> dhx.i18n.setLocale
  • vault.setURL -> vault.uploader.config.target
  • vault.upload -> vault.uploader.send

Changed events

  • vault.attachEvent("onBeforeFileAdd", function(){}) -> vault.events.on("BeforeAdd",function(){})
  • vault.attachEvent("onBeforeFileRemove",function(){}) -> vault.events.on("BeforeRemove",function(){})
  • vault.attachEvent("onBeforeFileUpload",function(){}) -> vault.events.on("BeforeUploadFile",function(){})
  • vault.attachEvent("onUploadCancel",function(){}) -> vault.events.on("Cancel",function(){})
  • vault.attachEvent("onUploadComplete",function(){}) -> vault.events.on("UploadComplete",function(){})
  • vault.attachEvent("onUploadFail",function(){}) -> vault.events.on("UploadFail",function(){})
  • vault.attachEvent("onUploadFile",function(){}) -> vault.events.on("UploadFile",function(){})
  • vault.attachEvent("onFileAdd",function(){}) -> vault.events.on("AfterAdd",function(){})
  • vault.attachEvent("onFileRemove",function(){}) -> vault.events.on("AfterRemove",function(){})

Removed API

vault.addDraggableNode vault.readableSize vault.setSLURL
vault.disable vault.removeDraggableNode vault.setSWFURL
vault.enable vault.setAutoRemove vault.setSizes
vault.getData vault.setDownloadURL vault.setSkin
vault.getFileExtension vault.setFilesLimit vault.setWidth
vault.getMaxFileSize vault.setHeight vault.unload
vault.getSLVersion vault.setMaxFileSize
vault.getStatus vault.setProgressMode

Removed events

  • vault.attachEvent("onBeforeClear",function(){})
  • vault.attachEvent("onClear",function(){})
  • vault.attachEvent("onDrop",function(){})
