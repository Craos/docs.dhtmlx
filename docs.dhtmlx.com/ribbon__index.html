Ribbon

dhtmlxRibbon combines features of both dhtmlxToolbar and dhtmlxTabbar components. It provides users with means of easy access to commands and tools used in an application or in a website by organizing them into logical blocks. Thus, Ribbon is irreplaceable if you need to create an intuitive and user-friendly interface.
